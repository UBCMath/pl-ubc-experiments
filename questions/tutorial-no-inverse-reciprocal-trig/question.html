In PrairieLearn, while $\arcsin$, $\arccos$, and $\arctan$ are supported, $\operatorname{arccot}$, $\operatorname{arcsec}$, and $\operatorname{arccsc}$ are not.

In the box below:

  1. Try typing {{params.inv}}(x) or {{params.alt}}(x). Notice the answer is invalid.
  2. Try typing in an equivalent trigonometric function that uses only arcsin(), arccos(), and/or arctan().
