In PrairieLearn, there is a difference between an invalid input and an incorrect input.

Different types of inputs have different rules about this.

  1. Type an “$x$” in each of the boxes below and press “Save & Grade”. Think about the feedback you receive.
  2. Click the question mark buttons near the input boxes - they are your friend!
  3. Type “$1$” in each of the boxes and press “Save & Grade”. Think about how this feedback is different from what you received in part a.
  4. Type “1/2” in each of the boxes and press “Save & Grade”. Think about the feedback. You should also explore the grey “Why?” buttons that appear after you submit.
  5. Type “0.6” in each of the boxes and press “Save & Grade”. Think about the feedback. Again, explore the grey “Why?” buttons that appear.

Applying what we’ve learned

What is $\displaystyle \frac { {{params.a}} } { {{params.b}} }$? Enter an appropriate answer into each box: you may need to format or round your answers appropriately.

Bonus: experiment with leaving some of the boxes blank: this is usually invalid but sometimes the question-author will configure this to simply be wrong.

One important difference to note is that an invalid response does not use up any of our “attempts”, but an incorrect response does.


In summary, we have learned that PairieLearn can be very particular about how we format our answers. We saw a contrast between invalid answers and incorrect answers, and learned a bit about “debugging” our submissions.
